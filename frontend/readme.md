# Frontend

Este é o frontend desenvolvido com **React**, **Vite** e **TypeScript**.  

## Requisitos

Antes de começar, você precisa ter instalado:

- Node.js (versão 18 ou superior)
- npm ou yarn

## Instalação do projeto  

Clone o repositório e entre na pasta do frontend:  

```react
cd frontend
```  

Instale as dependências:  

```react
npm install
``` 

Executar o projeto em modo desenvolvimento  

```react
npm run dev
```  

O servidor será iniciado em:  

http://localhost:5174
